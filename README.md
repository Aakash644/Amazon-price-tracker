# Amazon-price-tracker
## Developed a amazon price tracker program to track the prices of your favorite product and send the user notifications through email.
![amazon](https://user-images.githubusercontent.com/92630714/218726465-9706d282-a5b6-462f-9111-d494629df512.png) 
## Tech stack used: Python, Beautiful Soup, Smtplib, requests
## Note: Passing headers with parameters such as Accept language and User agent is <br>compulsory in order to access the actual Real time data.
## Go to http://myhttpheader.com/ to get your browser Headers(parameters).
## Get the url of your favourite amazon product from their website.
## Enter your expected price if the original price is less than or equal to expected price, you will get a required notification.
